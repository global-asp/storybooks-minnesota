Dharka iskuulka School clothes Les vêtements d'école

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini

Abdi Muse

Somali

Level 1

Dharkan wuu dheeryahay

This dress is long.

Cette robe est longue.

Funaanadan waa mid wayn.

This jersey is big.

Ce pull est grand.

Boorsadan waa mid wayn.

This bag is big.

Ce sac est grand.

Suunkan waa...

This belt is...

Cette ceinture est...

Koofiyadan waa mid yar.

This hat is small.

Ce chapeau est petit.

Sharaabaadaha waa kuwo gaaban.

The socks are short.

Les chaussettes sont courtes.

Laakiin kabahani waa kuwo cusub.

But these shoes are new.

Mais ces souliers sont neufs.

... si fiican bey ula egyihiin.

... they fit well.

... ils font bien.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini
Translated by: Abdi Muse
